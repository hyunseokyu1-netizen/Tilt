# Google Play 스토어 등록 정보 — 한국어

## 앱 이름 (50자 이하)
TILT: 미로 퍼즐

## 짧은 설명 (80자 이하)
기기를 기울여 3×3 그리드를 탐색하세요. 시간과의 싸움! 글로벌 순위에 도전!

## 전체 설명 (4000자 이하)

TILT는 시각장애인을 포함한 모든 이들이 즐길 수 있도록 설계된 센서 기반 퍼즐 게임입니다.

기기의 가속도계를 활용해 3×3 좌표 그리드(A1–C3)에서 마커를 이동시키세요. 매 라운드 타이머가 만료되기 전에 목표 칸에 도달하세요. 성공할수록 타이머는 빨라져 당신의 한계를 시험합니다.

🎮 게임 방법
• 기기를 기울여 마커를 이동시키세요
• 빛나는 목표 칸에 시간이 다 되기 전 도달하세요
• 라운드마다 타이머가 짧아집니다 — 얼마나 버틸 수 있을까요?

🔊 접근성 중심 설계
TILT는 화면을 보지 않고도 완벽하게 플레이할 수 있습니다:
• 이동마다 좌표 음성 안내 (예: "B2에서 A1로")
• 시간이 줄어들수록 60~180 BPM으로 빨라지는 동적 메트로놈
• 이동, 벽 충돌, 라운드 완료 시 햅틱(진동) 피드백
• 음성 안내 ON/OFF 토글 — 원하는 방식으로 플레이

🏆 글로벌 경쟁
• 점수를 등록하고 실시간 글로벌 리더보드에 도전
• 동점자 처리를 지원하는 DENSE 순위 시스템
• 내 순위와 바로 주변 플레이어 확인 (±3행)

📍 좌표 시스템
3×3 그리드의 각 칸에는 A1–C3 고유 좌표가 부여됩니다. 라운드 시작 전 음성으로 현재 위치와 목표를 안내하여, 시각 없이도 완벽한 플레이가 가능합니다.

캐주얼 게이머부터 상위권을 노리는 도전자까지, TILT는 반응 속도·공간 인지력·집중력을 한 칸씩 시험합니다.
